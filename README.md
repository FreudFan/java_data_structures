# 简介
#### 该项目是基于 Java 的关于部分数据结构的代码实现。

## 结构

* Array —— 数组  
* AVL_Tree —— AVL树  
* BST —— 二分搜索树  
* HashTable —— 哈希表(链地址法,基于红黑树)  
* LinkedList —— 链表  
* Map —— 映射(基于BST,链表实现)  
* MaxHeep —— 最大堆,基于堆的优先队列
* RedBlackTree —— 红黑树(左倾红黑树)  
* SegmentTree —— 线段树  
* Set —— 集合(基于BST,链表实现)  
* Trie —— 字典树  
* UnionFind —— 并查集
